在前面[blog](https://www.big-yellow-j.top/posts/2025/02/27/LLM-OUT.html)中介绍了llm生成策略，我们可以直接通过不同的采样策略/温度系数取控制模型输出，比如说在vllm中使用支持[各类参数](https://docs.vllm.ai/en/v0.6.4/dev/sampling_params.html)，但是有些时候需要严格控制模型输出内容，比如说强制模型必须输出json格式内容，最简单控制方法**直接使用prompt进行控制**，比如说在对应的prompt的**提示词中直接给出模型需要输出的一个例子**，让模型取模仿输出。但是有些时候这种方法不一定有效下面给出几种从模型上进行控制（从简单到复杂）。
## 基于模型原生控制
在调用模型生成过程中，有些模型是直接支持json格式输出的比如说（使用豆包模型可以使用openai方式进行调用）：
```python
from openai import OpenAI
import os

client = OpenAI(
    base_url="https://ark.cn-beijing.volces.com/api/v3",
    api_key="636b7b0b-2fe5-477d-9caf-ede47e5ab26e",
)

json_schema = {
    "type": "json_schema",
    "json_schema": {
        "name": "trump_analysis",
        "schema": {
            "type": "object",
            "properties": {
                "name": {"type": "string"},
                "age": {"type": "integer"},
                "birth_date": {"type": "string"},
                "world_wealth_rank": {"type": "integer"},
                "net_worth": {"type": "string"},
                "occupations": {"type": "array", "items": {"type": "string"}}
            },
            "required": ["name", "age", "world_wealth_rank"]
        }
    }
}
completion = client.chat.completions.create(
    model="doubao-seed-1-6-251015",
    messages = [
        {"role": "user", "content": "分析一个特朗普的生平，如年龄、世界富豪排名，按照json格式输出"},
    ],
    # response_format={"type": "json_object"}
    # response_format={"type": "text"}
    response_format=json_schema
)
print(completion.choices[0].message.content)
```
只需要在模型输出过程中[进行指定](https://developers.openai.com/api/reference/resources/chat/subresources/completions/methods/create#(resource)%20chat.completions%20%3E%20(method)%20create%20%3E%20(params)%200.non_streaming%20%3E%20(param)%20response_format%20%3E%20(schema))： `response_format={"type": "json_object"}` 即可控制模型按照json格式输出，除此之外可以使用 `json_schema` 去控制让模型去填写字段到我的要求格式中。本质上是constrained decoding（约束解码）：在每一步生成下一个token时，动态屏蔽（mask）不符合格式的token，只从合法候选集中采样。
## 使用guidance控制模型输出
如果使用开源模型（基座模型能力没有调用API那么强）那么为了控制模型输出可以使用guidance等进行控制，最简单的方式假如模型是使用vllm进行部署
### 基于vllm方式
在vllm中主要是两种方式进行输出控制，**第一种和最上面的方式相同直接本地启动vllm服务**：
```bash
export TMPDIR=/root/autodl-tmp/hf_cache # 指定模型下载缓存路径
export TEMP=/root/autodl-tmp/hf_cache
export TMP=/root/autodl-tmp/hf_cache

vllm serve \
    Qwen/Qwen3-0.6B \
    --host 0.0.0.0 \
    --port 8001 \
    --gpu-memory-utilization 0.93 \
    --max-model-len 16384 \
    --max-num-seqs 256 \
    --max_lora_rank 128 \
    --trust-remote-code \
    --served-model-name qwen3-0.6B
```
，而后在调用时候直接用相同方式即可：
```python
from openai import OpenAI
client = OpenAI(
    base_url="http://localhost:8001/v1",
    api_key="vllm",
)

messages = [
    {"role": "system", "content": "You are a helpful assistant."},
    {"role": "user", "content": "分析一个特朗普的生平，如年龄、世界富豪排名，按照json格式输出"}
]
chat_completion = client.chat.completions.create(
    messages=messages,
    temperature=0.7,
    max_tokens=2048,
    extra_body={
        "repetition_penalty": 1.05,
    }
)
output = chat_completion.choices[0].message.content
print(output)
```
